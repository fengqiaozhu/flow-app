# Flow UI/UX 详细设计文档 (MVP)

**版本**: 1.0
**日期**: 2026-03-16
**状态**: 草案
**设计语言**: 瑞士极简编辑风格 (Swiss Minimal Editorial) + HarmonyOS NEXT 6.0 设计规范

---

## 1. 设计系统 (Design System)

### 1.1 色彩与质感 (Colors & Textures)
- **核心色板**:
    - **背景 (Background)**:
        - 浅色: `#FFFFFF` (纯白)
        - 深色: `#000000` (纯黑)
    - **内容背景 (Surface)**:
        - 浅色: `#F5F5F5` (浅灰，用于卡片/侧边栏)
        - 深色: `#121212` (深灰，用于卡片/侧边栏)
    - **强调色 (Accent)**:
        - 品牌蓝: `#007DFF` (HarmonyOS 标准蓝)
        - 收藏金: `#FFB400`
    - **文字 (Text)**:
        - 主标题: `#0F172A` (Light) / `#F1F5F9` (Dark)
        - 次要文字: `#64748B` (Light) / `#94A3B8` (Dark)
- **模糊视效**:
    - 导航栏/工具栏: `BlurStyle.COMPONENT_ULTRA_THIN` (动态高斯模糊)

### 1.2 排版规范 (Typography)
- **字体**: 统一使用 `HarmonyOS Sans`
- **层级**:
    - **大标题 (Display)**: `32fp`, Bold, 用于详情页封面
    - **标题 (H1)**: `20fp`, Bold, 用于列表标题
    - **副标题 (H2)**: `16fp`, Medium, 用于摘要/分类
    - **正文 (Body)**: `16fp`, Regular, 行高 `1.8fp`, 用于阅读区
    - **标注 (Caption)**: `12fp`, Regular, 用于时间/作者

### 1.3 关键动效 (Motion)
- **转场**: 共享元素转场 (Shared Element Transition)，列表项封面图 -> 详情页顶部封面。
- **反馈**: 
    - 侧滑删除/已读: 阻尼感振动反馈。
    - 点击卡片: 轻微缩放 (Scale) 配合系统标准涟漪。

## 2. 核心页面拆解 (Page Breakdown)

### 2.1 MainPage (主容器)
- **目标**: 提供全局导航切换，适配不同屏幕尺寸。
- **布局**:
    - **手机端**: 底部 `Tabs` (订阅、发现、收藏、设置)。
    - **平板/折叠屏**: 侧边 `SideBar` 导航，支持自定义图标。
- **交互**:
    - 沉浸式状态栏适配，状态栏颜色随页面背景自动切换。

### 2.2 EntryListPage (文章列表页)
- **目标**: 高效展示订阅内容，支持多视图切换。
- **UI 细节**:
    - **顶部 Tab**: 分类切换 (全部、技术、设计...)。文字加粗选中，无背景块。
    - **视图 1: 卡片模式 (Card Mode)**:
        - 16:9 比例封面图，图上蒙层显示 RSS 源图标。
        - 标题下方显示“作者 · 时间”，左对齐。
    - **视图 2: 列表模式 (List Mode)**:
        - 极简单行/双行展示，右侧 1:1 比例小缩略图。
    - **空状态**: 极简插画 + “去发现新的订阅源”按钮。
- **交互**:
    - 右滑: 标记已读 (蓝色勾选图标)。
    - 左滑: 稍后阅读 (黄色星标图标)。

### 2.3 EntryDetailPage (文章详情页)
- **目标**: 极致沉浸的深度阅读体验。
- **布局**:
    - **Hero Area**: 视差滚动封面图 + 悬浮大标题。
    - **Body Area**: 纯白/纯黑背景，单栏排版，无边距 (Full Width) 或 16vp 安全边距。
    - **Bottom Bar**: 悬浮毛玻璃工具栏 (返回、收藏、分享、字体设置)。
- **交互**:
    - 滚动隐藏标题栏，顶部状态栏渐显文章标题。
    - 双击文字: 切换到“纯净模式”（隐藏所有辅助 UI）。

### 2.4 SubscriptionPage (订阅管理页)
- **目标**: 树形展示分类与订阅源，支持一键刷新。
- **UI 细节**:
    - 采用 `List` 的 `Group` 模式。
    - 文件夹图标 (带颜色标识) + 订阅源图标 (圆形)。
    - 未读数采用红色气泡或简单的数字标注在右侧。
- **交互**:
    - 长按订阅源: 进入编辑模式（重命名、移动、删除）。

### 2.5 DiscoverPage (发现页)
- **目标**: 引导用户添加新内容。
- **UI 细节**:
    - 顶部巨大的搜索框 (Search Inset)。
    - 推荐列表 (Bento Grid 风格)，分类展示热门 RSS 源 (如：Tech, Design, News)。
    - 支持扫码添加 (集成系统标准扫码组件)。

### 2.6 SettingsPage (设置页)
- **目标**: 高度可定制化的系统设置。
- **UI 细节**:
    - 分组卡片 (Grouped Lists)。
    - 切换按钮、滑块 (控制字号)、选择器。
- **配置项预览**:
    - 阅读背景色、字体缩放实时预览小窗。

## 3. 响应式适配策略 (Responsive Strategy)

| 屏幕宽度 (Breakpoint) | 导航模式 | 列表列数 | 内容区宽度 |
|---|-|---|-|
| < 600vp (手机) | 底部 Tab | 1 列 | 100% |
| 600vp ~ 840vp (折叠屏) | 侧边栏 (Collapsed) | 1 ~ 2 列 | 80% (居中) |
| > 840vp (平板/桌面) | 侧边栏 (Expanded) | 2 ~ 3 列 | 固定 720vp (黄金阅读宽度) |

---

## 4. 辅助功能 (A11y)
- **字体缩放**: 适配系统字体大小调节，页面布局不乱。
- **高对比度**: 浅色模式下文本对比度 >= 4.5:1。
- **语义化**: 为所有图片提供 `alt` 描述，为按钮提供 `accessibleLabel`。
